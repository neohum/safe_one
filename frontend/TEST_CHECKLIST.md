# 안전한줄 React 앱 테스트 체크리스트

- **개발 서버**: http://localhost:5175/
- **메인 페이지**: http://localhost:5175/
- **선택 페이지**: http://localhost:5175/select

## ✅ 테스트 항목

### 1. 메인 페이지 (/) 테스트

#### 1.1 페이지 로딩
- [ ] 페이지가 정상적으로 로드되는가?
- [ ] 헤더가 표시되는가? (🏠 홈, 🔄 랜덤, 📋 선택)
- [ ] 푸터가 표시되는가? (안전한줄.com, 기획/제작 정보)

#### 1.2 날짜 표시
- [ ] 현재 날짜가 한국어로 표시되는가? (예: 2025년 11월 16일)
- [ ] 요일이 한국어로 표시되는가? (예: 토요일)
- [ ] 날짜 텍스트가 주황색(#f97316)으로 표시되는가?
- [ ] 폰트 크기가 충분히 큰가?

#### 1.3 랜덤 영상 표시
- [ ] 페이지 로드 시 랜덤 영상이 표시되는가?
- [ ] 안전 수칙 내용이 큰 글씨로 표시되는가?
- [ ] YouTube 영상이 임베드되어 표시되는가?
- [ ] 영상이 자동 재생되는가?
- [ ] 영상 플레이어의 크기가 적절한가?

#### 1.4 다른 영상 보기 버튼
- [ ] "🔄 다른 영상 보기" 버튼이 표시되는가?
- [ ] 버튼 클릭 시 새로운 랜덤 영상이 로드되는가?
- [ ] 버튼 호버 시 색상이 변경되는가?
- [ ] 버튼이 파란색 배경을 가지는가?

#### 2.1 페이지 로딩

- [ ] 헤더와 푸터가 표시되는가?

#### 2.2 검색 기능
- [ ] 검색 입력창이 표시되는가?
- [ ] 검색어 입력 시 실시간으로 필터링되는가?
- [ ] 키워드로 검색이 가능한가?
- [ ] 내용으로 검색이 가능한가?
- [ ] 채널명으로 검색이 가능한가?
- [ ] 검색 결과가 없을 때 "검색 결과가 없습니다" 메시지가 표시되는가?

#### 2.3 영상 목록
- [ ] 모든 영상이 그리드 형태로 표시되는가?
- [ ] 각 영상 카드에 내용이 표시되는가?
- [ ] 각 영상 카드에 채널명이 표시되는가?
- [ ] 각 영상 카드에 플레이 타임이 표시되는가?

- [ ] 그리드가 3열로 표시되는가?

#### 2.4 영상 선택 및 재생
- [ ] 영상 카드 클릭 시 전체 화면으로 전환되는가?
- [ ] 선택한 영상의 내용이 큰 글씨로 표시되는가?
- [ ] YouTube 영상이 자동 재생되는가?
- [ ] "← 목록으로 돌아가기" 버튼이 표시되는가?
- [ ] 돌아가기 버튼 클릭 시 목록으로 돌아가는가?
